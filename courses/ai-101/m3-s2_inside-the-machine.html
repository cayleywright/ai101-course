Module 3

Inside the Machine

You've seen what happens when you press send. Now here's what's under the hood. Every AI tool — whether it's a chatbot, a recommendation engine, or an autonomous agent — is built from the same five layers. Different products package them differently, but the building blocks don't change.

Understanding these layers means you can look at any AI tool and ask the right questions: what's the interface? How does it connect to my systems? Which model is running underneath? Where does it get its knowledge? And where is all of this actually running?

Case study

Now map these five layers onto what Sarah and David are actually using. Where is their stack strong? Where are the gaps?

✂
Sarah
Salon owner · 11 years
  • Interface: Fresha booking app, Instagram, Square POS
  • Connections: None — each tool is an island
  • Intelligence: Basic AI features built into Fresha and Instagram
  • Knowledge: Client notes in a physical notebook
  • Infrastructure: All cloud, all vendor-managed
Sarah's biggest gap is connections. Her tools don't talk to each other.

David

  • Interface: ServiceM8, Xero, Microsoft 365, Deputy
  • Connections: Manual — Excel spreadsheets bridge the gaps
  • Intelligence: CRM with AI features the team doesn't use
  • Knowledge: Tribal knowledge in senior staff's heads
  • Infrastructure: Mix of cloud SaaS and on-prem SharePoint
David's biggest gap is knowledge. Critical information lives in people, not systems.

Reflection

Think about a tool you use every day. Can you identify which of the five layers it touches? Where's the gap — the layer that's missing or weak? That gap is usually where the most value is hiding.
